# Nave Industrial. Yecapixtla (2023)

Este proyecto muestra el diseño sísmico y eólico de una nave industrial de 6,000 m² ubicada en Yecapixtla, Morelos, conforme a NTC-2023 e IMCA.

## Software utilizado
- SAP2000 24
- Excel
- IMCA / NTC CDMX

## Contenido
- `Modelo. Nave Yecapixtla.$2k`: archivo del modelo estructural para ser importado.
- `Memoria de Cálculo.pdf`: memoria completa con hipótesis, combinaciones, diseño.
- `Nave. Yecapixtla.png`: Imagen del modelo.
